# Hogwats Typing Academy

[English README](./README.md)

Hogwats Typing Academy 是一个受《哈利波特》风格启发的打字练习小游戏，是我做给 8 岁女儿玩的。

她很喜欢霍格沃茨的魔法世界，所以我把普通的键盘练习做成了一个小小的魔法学院冒险：创建玩家、通过分院帽进入学院、练习魔法词汇、解锁更难的关卡，一边玩一边熟悉键盘。

## 玩法

1. 创建或选择一个玩家。
2. 告诉分院帽一点自己的特点，然后获得所属学院。
3. 开始打字挑战。
4. 按照屏幕上的单词准确输入。
5. 遇到不熟悉的词，可以使用提示。
6. 完成关卡后查看分数、打字速度、准确率和连续正确次数。
7. 持续练习，解锁更高难度的课程，并逐步掌握更多单词。

游戏进度会保存在浏览器本地，所以不同玩家可以保留自己的进度，下次继续练习。

## 功能

- 适合喜欢《哈利波特》风格的小朋友的魔法学院主题
- 分院帽分院
- 按七年故事线组织的 16 个课程，共 128 个精选词
- 记录每个单词的练习次数、错误数和掌握状态
- 词汇图鉴，可查看课程、单词、提示和掌握状态
- 词汇图鉴支持按关键词、掌握状态和词语分类筛选
- 完成每个学年最后一课后展示学年证书
- 错词复习模式，集中练习还没掌握的单词
- 完成最终挑战后显示毕业状态
- 单词提示和语音风格反馈
- 分数、WPM、准确率和连续正确次数统计
- 虚拟键盘显示，帮助孩子熟悉按键位置
- 支持多个本地玩家
- 使用浏览器本地存储，不需要账号或后端服务

## 本地运行

需要先安装 Node.js。

```bash
npm install
npm run dev
```

生产构建：

```bash
npm run build
```

预览生产构建：

```bash
npm run preview
```

## 部署

这个项目已经配置好 Vercel。

- 构建命令：`npm run build`
- 输出目录：`dist`
- 框架：Vite

在 Vercel 中导入 GitHub 仓库即可部署。
